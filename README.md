## Ok jak teraz to cudo postawić
0. Kup kąkuter
1. Zainstaluj Putina aka Rubina aka Ruby (conajmniej 2.7 weź daj ok?)
2. Stwórz plik .env
```
TOKEN=token
CHANNELID=id kanału
```
3. Uruchom `gem install discordrb dotenv`
4. Uruchom `ruby main.rb`
5. Enjoy
